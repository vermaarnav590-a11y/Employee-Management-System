# 👨‍💼 Employee Management System

A Python-based Employee Management System that allows users to efficiently manage employee records using SQLite. This project demonstrates CRUD (Create, Read, Update, Delete) operations with a persistent database and an easy-to-use command-line interface.

---

## 🚀 Features

- ➕ Add Employee
- 📋 View All Employees
- 🔍 Search Employee by ID
- ✏️ Update Employee Details
- ❌ Delete Employee
- 💾 SQLite Database Storage
- 🖥 User-Friendly CLI Interface

---

## 🛠 Technologies Used

- Python 3
- SQLite3
- VS Code
